# Modelagem de Interface - Sistema de Votação

## Objetivo
Escolher o mascote da universidade.

## Conteúdo
- Modelagem de tarefas
- Diagrama de organização das tarefas
- Descrição do fluxo do sistema

## Tarefas

1. Ler explicação das alternativas e como votar  
2. Iniciar a votação  
3. Votar  
   - 3.1 Visualizar opções de mascotes  
   - 3.2 Escolher um mascote  
4. Confirmação de voto  
   - 4.1 Revisar a escolha de mascote  
   - 4.2 Confirmar o voto  
5. Agradecimento por participação
